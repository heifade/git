用远程获取分支a，需要2个命令：
git fetch origin
git checkout -b a origin/a

查看本地及远程分支
git branch -a

删除本地分支a
git branch -d a

删除远程分支
git push origin --delete c

将本地的master分支推送到远程分支c上
git push origin master:c

用当前分支创建b1分支，并检出b1分支
git checkout -b b1
用c1分支创建b1分支，并检出b1分支
git checkout -b b1 c1

用当前分支创建b1分支
git branch b1
用c1分支创建b1分支
git branch b1 c1


将b1与master 分叉变成一条直线
git rebase master b1 // 将b1分支中的修复变基到master上, server 中的代码被“续”到了 master 后面。
git checkout master
git merge b1
